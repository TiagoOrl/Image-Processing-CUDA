
## Img filters processed in CUDA GPGPU ##
Img processing application that I've developed in C++ CUDA.
To build, just execute make file (tested in Pop!OS 20.04)

### Run ###
$ ./process [--sobel|--sobelBW] imgInput.jpg outputName
